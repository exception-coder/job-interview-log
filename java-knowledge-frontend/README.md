# Java 知识库前端

一个现代化的 Java 知识库展示平台，用于学习和复习 Java 面试知识点。

## 特性

- 🎨 现代化深色主题 UI
- 📚 系统化的知识分类
- 🔍 快速搜索功能
- 📱 响应式设计
- ⚡ 流畅的动画效果

## 技术栈

- Vue 3 + TypeScript
- Vite
- Markdown-it
- Vue Router

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 项目结构

```
src/
├── components/          # 组件
│   ├── Sidebar.vue     # 侧边栏导航
│   └── ContentViewer.vue # 内容展示
├── data/               # 数据
│   └── knowledgeTree.ts # 知识树结构
├── App.vue             # 根组件
├── main.ts             # 入口文件
└── style.css           # 全局样式
```
