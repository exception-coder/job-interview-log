# ✅Redis如何高效安全的遍历所有key

# 典型回答

在Redis中遍历所有的key，有两种办法，分别使用KEYS命令和SCAN命令。
​

KEYS命令用于查找所有符合给定模式的键，例如KEYS *会返回所有键。它在小数据库中使用时非常快，但在包含大量键的数据库中使用可能会阻塞服务器，因为它一次性检索并返回所有匹配的键。
